# Keel Logo

## Files

| File | Description |
|------|-------------|
| `keel-logo.svg` | Primary mark. Deep navy blades + teal accent. Use on light backgrounds. |
| `keel-logo-light.svg` | Inverse mark. White blades + teal accent. Use on dark backgrounds. |
| `keel-logo-favicon.svg` | Simplified monochrome mark (no teal accent). Use at 16-32px sizes. |
| `keel-logo-wordmark.svg` | Mark + "Keel" text, horizontal layout. |

## Colors

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `#1a2332` | Blade fill, wordmark text |
| Accent | `#0ea5a0` | Gap signal line |
| Surface (light) | `#ffffff` | Background, inverse blade fill |
| Surface (dark) | `#1a2332` | Background for inverse version |

## Design Rationale

- **Split keel**: Two asymmetric blade shapes with a vertical gap. Represents task isolation, the core concept of Keel.
- **Asymmetry**: Left blade is wider and taller. Prevents the mark from reading as a generic chevron.
- **Tapered tips**: Both halves taper to sharp points at the top, suggesting direction and precision.
- **Teal accent**: A thin vertical line in the gap. Suggests signal, data flow, something alive inside the container.
- **Favicon**: Drops the teal accent for reliability at small sizes. Pure shape, no detail.

## Size Guidelines

- **Full mark (keel-logo.svg)**: 32px and above. Teal accent becomes visible around 32px.
- **Favicon (keel-logo-favicon.svg)**: 16-32px. The simplified shape holds better at small sizes.
- **Wordmark (keel-logo-wordmark.svg)**: Headers, documentation, README. Minimum display width ~200px.

## Usage Notes

- The mark is designed for flat backgrounds. Avoid placing on gradient or image backgrounds.
- Minimum clear space around the mark: 1/4 of the mark height on all sides.
- For the wordmark, the system font is a placeholder. Convert "Keel" text to outlines before production use, or replace with your chosen typeface.
